# 主题洞察追踪器

检测用户的洞察表达，自动沉淀到 L3 主题层，建立证据链与块引用。

---

## 触发词

- **发现类**："我发现...", "我意识到...", "突然明白..."
- **决策类**："我决定...", "我要...", "从今以后..."
- **能力类**："我擅长...", "我不擅长...", "我的优势是..."
- **顿悟类**："原来如此", "顿悟", "豁然开朗"

---

## 执行流程

### Step 1: 提取洞察

从用户表达中提取：
- **洞察类型**：发现 / 决策 / 能力认知 / 情绪觉察
- **主题关键词**：拖延、能量管理、决策、沟通等
- **具体内容**：用户的原话或提炼
- **来源位置**：当前对话的上下文（用于块引用）

### Step 2: 匹配或创建 L3 主题

**主题匹配规则：**
```python
theme_keywords = {
    "克服拖延": ["拖延", "不想开始", "懒得动", "任务启动"],
    "克服完美主义": ["完美主义", "做不好", "害怕失败", "完美"],
    "能量管理": ["精力", "能量", "疲惫", "状态好", "睡眠"],
    "决策质量": ["决定", "选择", "权衡", "纠结", "后悔"],
    "沟通表达": ["表达", "沟通", "说不清楚", "对话"],
    "技术成长": ["学习", "掌握", "熟练", "新技术"],
    "情绪觉察": ["焦虑", "压力", "平静", "情绪波动"]
}
```

**执行逻辑：**
1. 检查 `L3_User/02-Themes/` 是否已有匹配主题
2. **存在** → 追加到该主题的 "## 证据链" 章节
3. **不存在** → 创建新主题文件，初始化结构

### Step 3: 创建/更新主题文件

**新主题文件模板：**
```markdown
---
type: theme
created: YYYY-MM-DD
updated: YYYY-MM-DD
level: 1-5
category: growth | career | health | relationship
links:
  out: []        # 关联的其他主题
  in: []         # 谁指向我（自动维护）
---

# {{主题名}}

> 一句话定义这个主题的核心含义

## 核心洞察

| 日期 | 洞察 | 来源 |
|------|------|------|
| YYYY-MM-DD | [[2026-03-19#^insight-id\|具体洞察内容]] | 对话 |

## 证据链

### YYYY-MM-DD
**触发场景**：当时发生了什么
**洞察内容**：用户的原话或提炼
**行动改变**：基于这个洞察做了什么不同

## 相关主题

- [[相关主题A]] - 关联说明
- [[相关主题B]] - 关联说明

## 实践应用

- [ ] 待实践的行动1
- [x] 已实践的行动2（日期）
```

**更新现有主题：**
1. 在 "## 证据链" 下追加新条目
2. 添加块引用链接：`[[YYYY-MM-DD#^insight-id]]`
3. 更新 `updated` 字段

### Step 4: 添加块引用到 Daily（如当天有日报）

如果 `L1_Episodic/01-Daily/YYYY-MM-DD.md` 存在：
1. 在日报中找到对应洞察的位置
2. 添加块 ID：`^insight-id`（如 `^procrastination-1`）
3. 在主题文件中引用：`[[2026-03-19#^procrastination-1]]`

### Step 5: 调用 record skill（显式联动）

```python
Skill({
  skill: "record",
  args: "--mode insight --theme {{主题名}} --content '{{洞察摘要}}'"
})
```

## 块引用规范

**格式：** `[[文档名#^块ID]]`

**块ID命名规则：**
- 主题缩写 + 序号（如 `^proc-1`, `^energy-3`）
- 日期 + 序号（如 `^0319-1`）

**示例：**
```markdown
<!-- 在 Daily 中 -->
今天工作卡了很久，[[克服拖延#^proc-5\|我发现拖延的根源是完美主义恐惧]]。

<!-- 在主题中 -->
| 日期 | 洞察 |
|------|------|
| 2026-03-19 | [[2026-03-19#^proc-5\|拖延根源是完美主义恐惧]] |
```

## 主题层级定义

| Level | 含义 | 示例 |
|-------|------|------|
| 1 | 初步觉察 | 第一次意识到某个模式 |
| 2 | 反复验证 | 多次观察到同一现象 |
| 3 | 形成概念 | 能清晰描述这个模式 |
| 4 | 有方法论 | 知道如何应对/利用 |
| 5 | 内化为习惯 | 无需思考自动执行 |
