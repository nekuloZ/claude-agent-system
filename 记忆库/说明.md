## 四层架构

```
记忆库/
├── L0_工作区/        ← 工作层（短期，7日滚动）
├── L1_长期记忆/      ← 长期层（日志、项目、周报）
├── L2_知识库/        ← 知识层（流程、参考文档）
└── L3_用户档案/      ← 用户层（身份、主题、学习）
```

## 层级说明

| 层级 | 名称 | 内容 | 保留时间 | 谁来写 |
|------|------|------|----------|--------|
| L0 | 工作区 | 当日记录、待办、习惯 | 7天滚动 | AI 自动 |
| L1 | 长期记忆 | 日志、项目、周报 | 永久 | AI + 用户 |
| L2 | 知识库 | 系统文档、流程参考 | 永久 | AI 为主 |
| L3 | 用户档案 | 身份、课题、学习 | 永久 | 用户为主 |

## 快速定位

- "今天做了什么？" → `L0_工作区/7日记录.md`
- "项目 X 进展？" → `L1_长期记忆/02-项目/X.md`
- "系统怎么用？" → `L2_知识库/核心文档/`
- "我是谁？" → `L3_用户档案/01-身份/`
