# L0 工作区

## 作用
当前工作窗口，存放**今日详情 + 近7天归档摘要**。

- **TTL**：7天滚动，超过7天的内容自动归档到 L1
- **写入**：主要由 AI 自动写入
- **读取**：每次会话启动时自动加载

## 文件说明

| 文件 | 用途 | 格式 |
|------|------|------|
| `7日记录.md` | 今日详细记录 + 近7天摘要 | Markdown |
| `待办看板.md` | 当前待办事项列表 | Markdown（任务列表） |
| `习惯数据.json` | 习惯追踪数据（睡眠、洗澡等） | JSON |

## 7日记录.md 格式

```markdown
# 7日记录

## 今日（2026-03-20）
- [事件] 做了什么事情
- [决策] 做了什么决定
- [洞察] 有什么发现

## 近7天摘要
- 2026-03-19: [一句话总结]
- 2026-03-18: [一句话总结]
...
```
