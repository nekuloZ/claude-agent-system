# 记忆检索策略
_最后更新：2026-03-20_

本文档说明 memory_loader 的渐进式记忆加载策略。

---

## 渐进式加载流程

```
用户提问 / 会话启动
    │
    ▼
[Stage 0] L0 自动加载（~50ms）
    │ - 7日记录.md
    │ - 待办看板.md
    │ - 习惯数据.json
    │ - .current_task.json
    │
    ▼
[Stage 1] 意图识别
    │ - 提取关键词（项目名、人名、日期）
    │ - 确定需要加载的层级
    │
    ▼
[Stage 2] 提及层预加载（~200ms）
    │ - 路径匹配：02-项目/、01-日志/
    │ - 双链预加载：提取 [[wikilinks]] 关联笔记
    │
    ▼ 质量足够？
    ├── 是 → 返回结果 ✅
    │
    └── 否 → [Stage 3] 语义搜索（~1.2s）
               - 向量搜索补充
               - 返回 Top-K 相关片段
```

---

## Stage 0: L0 必载

**触发时机**：每次会话开始

**加载文件**：
| 文件 | 用途 |
|------|------|
| `记忆库/L0_工作区/7日记录.md` | 近7天工作摘要 |
| `记忆库/L0_工作区/待办看板.md` | 当前待办事项 |
| `记忆库/L0_工作区/习惯数据.json` | 习惯追踪数据 |
| `记忆库/L0_工作区/.current_task.json` | 当前任务上下文 |

---

## Stage 1: 意图识别

**分析用户 Query**：
- "项目A进展如何？" → 加载 `L1_长期记忆/02-项目/项目A.md`
- "昨天做了什么？" → 加载 `L1_长期记忆/01-日志/YYYY-MM-DD.md`
- "我擅长什么？" → 加载 `L3_用户档案/01-身份/`

**关键词提取**：
- 项目名称
- 人名
- 日期（昨天、上周、2026-03）
- 主题标签

---

## Stage 2: 提及层预加载

### 路径匹配

根据关键词匹配文件路径：
```
"项目A" → 记忆库/L1_长期记忆/02-项目/项目A.md
"日志"  → 记忆库/L1_长期记忆/01-日志/
"主题"  → 记忆库/L3_用户档案/02-主题/
```

### 双链预加载

读取文件中的 `[[wikilinks]]`，预加载关联笔记：

```markdown
# 项目A.md

使用了 [[技术方案]]，参考 [[文件路径规范]]。

→ 预加载:
- 技术方案.md
- 文件路径规范.md
```

---

## Stage 3: 语义搜索（可选）

**触发条件**：Stage 2 结果不足（匹配度 < 0.7 或结果 < 3 条）

**实现方式**：
- 使用 Gemini Embedding 向量搜索
- 搜索历史日志、项目文档
- 返回最相关的 Top-K 片段

**熔断保护**：
- 超时限制：5秒
- 失败时降级返回 Stage 2 结果

---

## Token 管理

### 估算规则

| 内容类型 | Token 估算 |
|----------|-----------|
| 中文字符 | 0.6 token/字 |
| 英文单词 | 1.3 token/词 |
| 其他字符 | 0.5 token/字符 |

### 加载策略

- **摘要模式**：只读前 500 字符
- **标题模式**：只读章节标题
- **全文模式**：完整读取（小文件）

---

## 优化建议

### 提高记忆检索质量

1. **使用清晰的文件名**
   ```markdown
   # 好
   [[用户画像分析]]
   [[技术架构设计]]

   # 不好
   [[aaa]]
   [[笔记1]]
   ```

2. **建立双链网络**
   ```markdown
   本项目参考了 [[设计模式]] 和 [[最佳实践]]。
   ```

3. **使用标签分类**
   ```markdown
   #项目A #工作 #后端
   ```

4. **定期归档整理**
   - 日复盘：整理 L0 → L1
   - 周复盘：清理过期临时文件

---

## 故障排查

### 记忆加载失败

**症状**："我找不到之前的记录"

**检查项**：
- [ ] L0 文件是否存在
- [ ] 文件路径是否正确（中文路径）
- [ ] 文件编码是否为 UTF-8
- [ ] 是否有 `AI-TODO` 未处理

### 双链无法识别

**症状**：`[[链接]]` 显示为普通文本

**原因**：
- 目标文件不存在
- 路径错误
- 中文文件名编码问题
