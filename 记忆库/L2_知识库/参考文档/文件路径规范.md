# 文件路径规范
_最后更新：2026-03-20_

本文档说明记忆库的文件路径规范和使用方法。

---

## 四层级架构

```
记忆库/
├── L0_工作区/        # 工作层（TTL：小时级）
├── L1_长期记忆/      # 长期层（TTL：天-月）
├── L2_知识库/        # 知识层（TTL：月-永久）
└── L3_用户档案/      # 用户层（TTL：永久）
```

---

## 各层详细路径

### L0_工作区

**路径**：`记忆库/L0_工作区/`

**用途**：当前会话的临时工作记录

**核心文件**：
| 文件 | 用途 |
|------|------|
| `7日记录.md` | 今日详情 + 近7天归档摘要 |
| `待办看板.md` | 待办事项列表 |
| `习惯数据.json` | 习惯追踪数据 |
| `.current_task.json` | 当前任务上下文 |

---

### L1_长期记忆

**路径**：`记忆库/L1_长期记忆/`

**用途**：历史日志、项目记忆、周报复盘

**目录结构**：
```
L1_长期记忆/
├── 01-日志/              # 每日工作日志
│   └── YYYY-MM-DD.md
├── 02-项目/              # 项目记忆文档
│   └── {{项目名}}.md
└── 周报/                 # 周复盘记录
    └── YYYY-Www.md
```

---

### L2_知识库

**路径**：`记忆库/L2_知识库/`

**用途**：流程文档、系统说明、参考资料

**目录结构**：
```
L2_知识库/
├── 核心文档/             # 系统架构、设计说明
├── 参考文档/             # 操作指南、最佳实践
└── 说明.md               # 本层说明
```

---

### L3_用户档案

**路径**：`记忆库/L3_用户档案/`

**用途**：用户自我定义、身份认同、成长记录

**目录结构**：
```
L3_用户档案/
├── 01-身份/              # 价值观、目标、偏好
├── 02-主题/              # 反复出现的人生课题
└── 03-学习/              # 技能成长记录
```

**重要原则**：AI 只读，不自动修改

---

## Skills 文件操作规范

### 必须使用 obsidian-markdown skill

```javascript
// 正确：使用 skill 写入
Skill:obsidian-markdown file_path="记忆库/L0_工作区/7日记录.md"

// 错误：直接使用 Write 工具
Write file_path="记忆库/L0_工作区/7日记录.md"  // 可能编码错误
```

### 编码规范

- 所有 Markdown 文件必须使用 **UTF-8 编码**
- 中文文件名支持（本系统使用中文路径）
- 路径分隔符使用正斜杠 `/`

| 个人偏好 | `记忆库/L3_用户档案/01-身份/` |
